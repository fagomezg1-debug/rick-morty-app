# Rick and Morty App

Aplicación desarrollada en Angular que consume la API pública de Rick and Morty.

## Descripción
Este proyecto muestra una lista de personajes de la serie Rick and Morty en tarjetas, consumiendo datos desde una API pública mediante `HttpClient`.

## Funcionalidades
- Consumo de API con HttpClient
- Listado de personajes
- Visualización de imagen, nombre, estado y especie
- Buscador por nombre
- Filtro por estado
- Manejo de carga
- Diseño personalizado con CSS

## Tecnologías utilizadas
- Angular
- TypeScript
- HTML
- CSS
- API pública de Rick and Morty

## API utilizada
`https://rickandmortyapi.com/api/character`
